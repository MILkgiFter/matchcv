# Локальная AI-модель на телефоне

## Как это работает

1. Пользователь устанавливает APK/IPA (не Expo Go)
2. Profile → **Local AI Model** → **Download** (~256 MB по Wi-Fi)
3. **Load model** — модель загружается в память
4. Chat работает **офлайн** без API-ключей

Модель: **SmolLM2 360M** (GGUF, квантизация Q4) — лёгкая open-source модель от Hugging Face.

## Почему не в Expo Go?

`llama.rn` использует нативный C++ (llama.cpp). Нужна сборка:

```powershell
npm run build:preview
```

Установите APK на телефон — там появится кнопка скачивания модели.

## Упаковать модель в APK?

Можно, но APK вырастет на **+256 MB**. Сейчас модель **скачивается после установки** — так принято в мобильных LLM-приложениях.

## Облако vs локально

| Функция | Локально | Облако (Groq/backend) |
|---------|----------|------------------------|
| Chat | ✅ SmolLM2 | ✅ Llama 70B / GPT |
| Анализ PDF | ❌ слабо | ✅ |
| Match job | ❌ | ✅ |
| Cover letter | ❌ | ✅ |

Для резюме и вакансий лучше оставить backend + Groq.
